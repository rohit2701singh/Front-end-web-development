Difference between inline, block and inline-block

Inline elements:

  1. respect left and right margins and padding, but not top and bottom
  2. cannot have a width and height set. It will not accept height and width. It will just ignore it.
  3. allow other elements to sit to their left and right.

Block elements:

  1. respect all of those
  2. force a line break after the block element
  3. acquires full-width if width not defined

Inline-block elements:

  1. Characteristics of block, but lays out inline
  2. allow other elements to sit to their left and right
  3. respect top and bottom margins and padding
  4. respect height and width

  • An inline element has no line break before or after it, and it tolerates HTML elements next to it.

  • A block element has some whitespace above and below it and does not tolerate any HTML elements next to it.

  • An inline-block element is placed as an inline element (on the same line as adjacent content), but it behaves as a block element.
