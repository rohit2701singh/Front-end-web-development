Display Property

The display property is used to specify how an element is shown on a web page.

Every HTML element has a default display value, depending on what type of element it is. The default display value for most elements is block or inline.

The display property is used to change the default display behavior of HTML elements. Setting the display property of an element only changes how the element is displayed, NOT what kind of element it is.

display property has many values: ex: inline, block, inline-block, flex, grid, none
