Css Transition

CSS transitions allows you to change property values smoothly, over a given duration.

for example if I want to change the background color of a box when I hover over it, then without a transition, it will be a sudden change from one color to another. But with a transition, it will be a smooth and gradual change from one color to another within a specified time.

Transitions enable you to define the transition between two states of an element. Different states may be defined using pseudo-classes like :hover or :active or dynamically set using JavaScript

You can apply transition to any CSS property that has intermediate values 

        transition
        transition-delay
        transition-duration
        transition-property
        transition-timing-function
