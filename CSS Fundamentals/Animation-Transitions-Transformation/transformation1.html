Css Transformation and Transition Property

CSS 2D Transforms Functions

CSS transforms allow you to move, rotate, scale, and skew elements. CSS transforms allows elements styled with CSS to be transformed in two-dimensional or three-dimensional space. 

        properties:
            translate():  moves an element from its current position;
                 (x-axis, y-axis)=> (50, 50) means right and down from current position
            
            rotate(): rotates an element clockwise or counter-clockwise according to a given degree
           
            scaleX(): increases or decreases the width of an element
            scaleY(): increases or decreases the height of an element
            scale():  increases or decreases the size of an element; scale(width, height)
            
            skewX():  skews an element along the X-axis by the given angle.
            skewY():  skews an element along the Y-axis by the given angle.
            skew(): skews an element along the X and Y-axis by the given angles.
           
            matrix(): The parameters are as follow: 
                matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())
