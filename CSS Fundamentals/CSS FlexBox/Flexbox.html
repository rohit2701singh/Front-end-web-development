Flexible Boxes

Since flexbox is a whole module and not a single property, it involves a lot of things including its whole set of properties. Some of them are meant to be set on the container (parent element, known as “flex container”) whereas the others are meant to be set on the children (said “flex items”). 

            The CSS properties we use for the flex container are:
                flex-direction: row | row-reverse | column | column-reverse
                flex-wrap: nowrap | wrap | wrap-reverse
                flex-flow: column wrap (shorthand property for setting both the flex-direction and flex-wrap properties)
                
                justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly(alignment along the main axis)
                align-items: stretch | flex-start | flex-end | center | baseline | first baseline | last baseline (alignment along the cross axis)
                align-content:  similar to align-items, but instead of aligning flex items, it aligns the flex lines.
                
                gap: 10px;  (spacing only between items not on the outer edges.)
                gap: 10px 20px; /* row-gap column gap */
                row-gap: 10px;
                column-gap: 20px;
        
            The CSS properties we use for flex items are:
                order: specifies the order of the flex items inside the flex container.(default 0)
                flex-grow: specifies how much a flex item will grow relative to the rest of the flex items.(default 0)
                flex-shrink: specifies how much a flex item will shrink relative to the rest of the flex items if necessary.(default 1)
                flex-basis: specifies the initial length of a flex item.
                flex: shorthand for flex-grow, flex-shrink and flex-basis combined (default is 0 1 auto)
                align-self: specifies the alignment for the selected item, overrides the default alignment set by the container's align-items property.
