  • Child combinator (>)
  • Next sibling combinator (+)

Pseudo Classes and Elements

A CSS pseudo-class is used to define a special state of an element. ex: Style an element when a user moves the mouse over it. note=> :nth-child() => : single colon 

            selector:pseudo-class {
                property: value;
              }

A CSS pseudo-element is used to style specific parts of an element. ex: Style the first letter or line, of an element. Styles the user-selected text note=> ::selection => :: double colon 

            selector::pseudo-element {
                property: value;
              }
